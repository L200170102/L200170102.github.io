UPDATE




Perintah ini digunakan untuk memperbaharui data lama menjadi data terkini. Jika anda memiliki data yang salah atau kurang update dengan kondisi sekarang, maka dapat diubah isi datanya dengan menggunakan perintah UPDATE.

Syntaxnya sebagai berikut:
UPDATE nama_tabel SET nama_field = data_baru
WHERE nama_field = data_lama;
